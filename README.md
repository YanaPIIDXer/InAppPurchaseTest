# InAppPurchaseTest
App内課金の実験用リポジトリ

# 作るもの

- ユーザの所持金的なものを課金で増やすとか
- 消耗型・非消耗型・自動更新サブスク・非自動更新サブスクの全パターンの検証

# 各タイプの役割

| Key                | Value                          |
| ------------------ | ------------------------------ |
| 消耗型             | 所持金を増やす                 |
| 非消耗型           | フラグを立てる                 |
| 自動更新サブスク   | ログボ解禁                     |
| 非自動更新サブスク | 消耗型課金による所持金増加量UP |
